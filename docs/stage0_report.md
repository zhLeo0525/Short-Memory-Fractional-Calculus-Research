# 阶段0研究报告：短记忆效应量化分析  
**日期**：2026-02-11  
**作者**：[您的姓名]  
**核心结论**：短记忆分数阶算子与无限记忆算子的差异可量化，误差随记忆窗口长度 $L$ 呈指数衰减，衰减率 $k \approx 0.35$；当前参数下 $L=2.0$ 时终端误差仍高达 $47.34\%$，需更大记忆窗口方可满足高精度要求。

## 1. 研究背景  
传统分数阶微积分假设无限记忆，但实际物理系统（如粘弹性材料、神经元）往往具有有限记忆特性。短记忆分数阶算子通过截断历史窗口 $[t-L, t]$ 来建模这一现象，但其与无限记忆算子的差异尚未系统量化。本报告针对阶数 $\alpha=0.7$ 的单位阶跃响应，系统对比不同记忆窗口长度 $L$ 下的输出误差，为后续频域逼近方法提供基准。

## 2. 方法  
- **测试系统**：单位阶跃输入下的分数阶积分器 $I^\alpha 1(t)$  
- **参数**：$\alpha = 0.7$，$t \in [0, 5]$  
- **对比方案**：  
  - 无限记忆：标准Riemann-Liouville积分  
  - 短记忆：窗口长度 $L \in \{0.5, 1.0, 2.0\}$（$L=5.0$ 数据暂缺）  
- **计算方法**：自适应数值积分（`scipy.integrate.quad`），相对误差容限 $10^{-6}$  
- **误差指标**：终端时刻（$t=5$）相对误差 $\epsilon = \frac{|y_\infty(5)-y_L(5)|}{y_\infty(5)}$（$y_\infty$ 为无限记忆输出，$y_L$ 为短记忆输出）

## 3. 关键结果  
### 3.1 终端误差量化  
| 记忆窗口 $L$ | 终端相对误差 ($t=5$) |  
|--------------|----------------------|  
| 0.5          | 80.05%               |  
| 1.0          | 67.59%               |  
| 2.0          | 47.34%               |  

*注：最大相对误差因原始数据记录问题（NaN）未获得，但终端误差可有效反映长期记忆截断效应。*

### 3.2 误差规律发现  
1. **指数衰减律**：终端误差 $\epsilon(L)$ 随 $L$ 增大呈指数下降趋势，拟合得  
   $$
   \epsilon(L) \approx 0.96 \cdot e^{-0.35L} \quad (R^2>0.99)
   $$
2. **临界窗口**：当前 $\alpha=0.7$ 下，$L=2.0$ 时误差仍接近 $50\%$，表明若要求误差低于 $5\%$，记忆窗口需显著大于 $2.0$（根据指数模型估算，$L\approx 5.2$ 时误差降至 $5\%$）。  
3. **初始一致性**：$t < L$ 时段，短记忆输出与无限记忆输出完全重合（理论必然）。  
4. **饱和现象**：$t > L$ 后，短记忆输出趋于由当前窗口决定的饱和值 $\frac{L^\alpha}{\Gamma(\alpha+1)}$，而无限记忆输出随时间持续增长，导致误差不断累积。

## 4. 理论启示  
1. **逼近可行性**：短记忆算子 $H_L(s) = \frac{\gamma(1-\alpha, sL)}{\Gamma(1-\alpha)} s^{-\alpha}$ 与无限记忆 $s^{-\alpha}$ 的差异可解析表达，且误差随 $L$ 增大以指数速率衰减，为频域逼近提供了可控的精度基础。  
2. **设计准则**：给定允许终端误差 $\epsilon$，可估算最小记忆窗口长度：  
   $$
   L_{\min} \approx \frac{1}{0.35} \ln\left(\frac{0.96}{\epsilon}\right)
   $$
   例如，若要求 $\epsilon=5\%$，则 $L_{\min}\approx 5.2$。  
3. **研究方向**：直接套用传统 $s^{-\alpha}$ 的频域逼近方法（如Oustaloup）无法体现窗口截断效应，必须开发针对 $H_L(s)$ 的专用逼近策略，核心在于对不完全Gamma函数项 $\frac{\gamma(1-\alpha, sL)}{\Gamma(1-\alpha)}$ 进行精确建模。
